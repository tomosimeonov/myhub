# Setup Guide

## Download MarKLogic 6

http://developer.marklogic.com/products

## Install MarkLogic 

###Windows x64

Standart Installation 

###RedHat x64

As the root user, install the package with the following command:
rpm -i /tmp/MarkLogic-6.0-1.amd64.rpm

## Start MarkLogic 

localhost:8000
